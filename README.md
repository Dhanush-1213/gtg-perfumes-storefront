# GTG Perfumes Assignment Starter

This is a vanilla HTML/CSS/JS starter build for the uploaded assignment.

## Included requirements
- Responsive header with hamburger menu on tablet/mobile
- Functional gallery with arrows, dots, and thumbnails
- Two radio groups with dynamic add-to-cart link updates
- Expandable subscription panels
- Count-up stats animation on scroll
- HTML comparison table
- Clean folder structure ready for submission zip

## Important note
The actual Figma design assets and exact copy were not embedded in the DOCX file. Replace the placeholder SVGs and text with the exported Figma assets/content to make it pixel-perfect.
